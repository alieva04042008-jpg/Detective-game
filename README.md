# Detective-game
The Detective - Murder Mystory Game

What is this?
The Detective is a full-stack web game built with a Node.js backend and a React frontend. Every game generates a brand new murder mystery using AI. Players interrogate three AI-powered suspects with a limited number of questions, gather clues, and submit their accusation. AI then rates their reasoning and reveals the truth. This is mini project made for fun :*)


How it works:
Player clicks "Start new case"
        -
AI generates a full mystery (victim, suspects, culprit, clues)
        -
Player interrogates 3 suspects — 3 questions each (9 total)
        -
Each suspect responds
        -
Clues unlock after each interrogation
        -
Player accuses a suspect and explains their reasoning
        -
AI scores the accusation and reveals what really happened
